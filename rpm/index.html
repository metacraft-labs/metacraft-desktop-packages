Welcome to the Metacraft Labs RPM repository

This repository is used for hosting all Metacraft Labs packages for applications such as CodeTracer and its tracers.

To use this repository use the following configuration to /etc/yum.repos.d/metacraft-rpms.repo :

[ude-rpms]
name=MetacraftRPM
baseurl=https://rpms.codetracer.com/
enabled=1
gpgcheck=0

Available packages:

  • codetracer - Provides CodeTracer, a user-friendly time-travelling debugger for a wide range of programming languages
